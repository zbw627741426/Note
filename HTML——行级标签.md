# HTML——行级标签

## 有哪些标签
- 文本格式化标签
- 超链接标签

        <a href="..." target="目标窗口（常用有：\_self \_blank">...</a>
        
- 图片标签

        <img src="..." alt="..." width='...' heigth="..." >

- `<span></span>`
- `<br />`
- 常用特殊符号

        空格 &nbsp
        大于号(>) &gt
        小于号(<) &lt
        引号(") &quot
        版权号 &copy
